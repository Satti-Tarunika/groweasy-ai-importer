\# GrowEasy AI CSV Importer



An AI-powered CSV Importer built using Next.js, Express.js, and Google Gemini AI.



\## Features



\- Upload CSV files

\- AI-powered column mapping using Google Gemini

\- CSV preview

\- Download mapped CSV

\- Modern responsive UI

\- Express backend

\- Next.js frontend



\## Tech Stack



\- Next.js

\- React

\- Express.js

\- Node.js

\- Tailwind CSS

\- Google Gemini API



\## Project Structure



```

groweasy-ai-importer/

│

├── frontend/

├── backend/

└── README.md

```



\## Installation



\### Backend



```bash

cd backend

npm install

npm run dev

```



\### Frontend



```bash

cd frontend

npm install

npm run dev

```
